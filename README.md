# pure-virtual-list

A pure virtual list component for React.

## Features

- [x] Virtual scrolling
- [x] Static item height
- [x] Custom item renderer
- [x] Custom item buffer count

## TODO in the future

- [ ] Dynamic item height

## Installation

```bash
npm install pure-virtual-list
```

## Usage

```jsx
import { VirtualList } from 'pure-virtual-list';
```
